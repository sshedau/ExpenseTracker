# Expense Tracker

A simple Expense Tracker application that allows users to manage and track their daily expenses.
Users can add, view, update, and delete expenses to keep track of their spending.

## Features

* Add new expenses
* View all expenses
* Update existing expenses
* Delete expenses
* Categorize expenses

## Tech Stack

* Java
* Spring Boot
* Spring Data JPA
* MySQL / PostgreSQL
* Maven

## API Endpoints

GET /expenses → Get all expenses
GET /expenses/{id} → Get expense by ID
POST /expenses → Add new expense
PUT /expenses/{id} → Update expense
DELETE /expenses/{id} → Delete expense

## How to Run

1. Clone the repository

```
git clone https://github.com/yourusername/expense-tracker.git
```

2. Go to the project folder

```
cd expense-tracker
```

3. Run the application

```
mvn spring-boot:run
```

4. Open in browser

```
http://localhost:8080
```
